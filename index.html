My Projects

Study Sage

The Study Sage project leverages advanced technology to predict students' performance in tests and exams, taking into account factors such as their family background, food habits, logic, and reading aptitude. By analyzing these variables, the project can provide educators with personalized insights into each student's learning capabilities, allowing them to tailor their instructional approach accordingly. With this technology, teachers can identify and address areas of weakness and optimize their students' performance potential. Ultimately, the Study Sage project aims to help students achieve academic success by providing teachers with valuable insights that can be used to enhance their teaching methods and foster a supportive learning environment.

Voice Assistant

I developed a Windows voice assistant project that allows users to interact with their computer using voice commands. Using speech recognition technology, the assistant can perform a variety of tasks, such as opening applications, searching the web, sending emails, and controlling the computer's settings. The project was built using Python and various libraries such as pyttsx3, SpeechRecognition, and PyAudio. Through this project, I gained experience in natural language processing, audio processing, and user interface design. Overall, this project showcases my ability to develop practical applications that can improve user experience and productivity.
